Detailed Description

SQLite3 persistence layer — users, jobs, and sessions.

Thin wrapper around libsqlite3. All operations use prepared statements. The database is opened once at startup (db_open()) and closed at shutdown (db_close()). WAL journal mode is enabled for concurrent read access.
