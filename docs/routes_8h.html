Detailed Description

HTTP request dispatcher and REST API handler interface.

Exposes a single Mongoose event-handler callback that routes incoming HTTP requests to the appropriate handler function and falls back to serving static files from the configured www directory.

Main Mongoose event handler — register as the HTTP listen callback.

Dispatches MG_EV_HTTP_MSG events to the correct REST endpoint handler based on HTTP method and URI pattern. Unmatched requests are served as static files from the configured www directory.
